Noise

human-readable graphic design

Noise is a concise, friendly language for designing graphics. It encourages readability, reuse, and resolution independence.

Noise translates directly to SVG 1.1, but adds the convenience of variables, functions, and transparent colors.

Noise is open-sourced under an MIT license.

  • Noise on GitHub

Noisepad is an editor application for writing graphics in Noise. It provides live visual feedback, code completion, gestural input, and more.

Noisepad is pre-alpha software and requires Mac OS 10.7 or later. It is not presently open-sourced.
